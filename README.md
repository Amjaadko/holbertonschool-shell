# io_redirections_and_filters

0-hello_world: Prints "Hello, World"
1-file_permission: Displays file permissions in current directory
2-hello_you: Prints "Hello," followed by current user
3-file_type: Displays the type of "my_file"
4-read_me: Displays the first 10 lines of "my_file"
5-count_i: Counts the number of lines in "my_file"
6-multiple_i: Counts number of files in current directory
7-file: Displays details of "hello"
8-save_last_lines: Displays last 10 lines of "my_file"
9-lets_move: Renames "hello" to "hello_bak"
10-remove_me: Removes "hello_bak"
